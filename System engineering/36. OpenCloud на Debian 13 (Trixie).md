- Сервер: https://github.com/opencloud-eu/opencloud#

## Инструкции
- Установка сервера: https://docs.opencloud.eu/docs/admin/getting-started/other/bare-metal
- Установка GO: https://go.dev/doc/install

## Трудности при сборке и их решение

### 1. Проблема с устаревшими зависимостями от OwnCloud

**Ошибка:**
```
get: protoc-gen-microweb.mod: getting github.com/owncloud/protoc-gen-microweb@v0.0.0-20220808092353-b5d6c3960e19
```

**Решение:**
```bash
# Удалить проблемные файлы
rm -f .bingo/protoc-gen-microweb.sum
rm -f .bingo/protoc-gen-microweb.mod

# Закомментировать строку в .bingo/variables.env
# PROTOC_GEN_MICROWEB="${GOBIN}/protoc-gen-microweb-v0.0.0-20220808092353-b5d6c3960e19"
```

### 2. Ошибка 403 Forbidden при генерации protobuf

**Ошибка:**
```
Failure: 403 Forbidden
make[1]: *** [../../.make/protobuf.mk:23: /root/programs/OpenCloud/Git/opencloud/protogen/buf.sha1.lock] Error 1
```

**Решение:**
```bash
# Закомментировать зависимости в protogen/proto/buf.yaml
# deps:
#   - buf.build/googleapis/googleapis
#   - buf.build/grpc-ecosystem/grpc-gateway
```

### 3. Отсутствие команды buf

**Ошибка:**
```
-bash: buf: command not found
```

**Решение:**
```bash
# Скачать buf вручную
wget https://github.com/bufbuild/buf/releases/download/v1.39.0/buf-Linux-x86_64
mv buf-Linux-x86_64 .bingo/buf-v1.39.0
chmod +x .bingo/buf-v1.39.0
ln -sf /root/programs/OpenCloud/Git/opencloud/.bingo/buf-v1.39.0 .bingo/buf
export PATH=$PATH:/root/programs/OpenCloud/Git/opencloud/.bingo
```

### 4. Отсутствие .bingo/Variables.mk

**Ошибка:**
```
Makefile:78: .bingo/Variables.mk: No such file or directory
```

**Решение:**
```bash
# Создать вручную
cat > .bingo/Variables.mk << 'EOF'
GOBIN := $(abspath .bingo)
GO := go
EOF
```

### 5. Ошибка команды optimize-grammar в search сервисе

**Ошибка:**
```
bash: line 1: optimize-grammar: command not found
```

**Решение:**
```bash
# Установить pigeon на локальной машине и скопировать на сервер
# На локальной машине:
go install github.com/mna/pigeon@latest

# На сервер:
scp /home/user/go/bin/pigeon root@opencloud:/root/go/bin/
chmod +x /root/go/bin/pigeon
```

### 6. Ошибка get: command not found в settings сервисе

**Ошибка:**
```
bash: line 1: get: command not found
```

**Решение:**
```bash
# Временно отключить проблемные сервисы
mv services/settings services/settings.disabled
mv services/search services/search.disabled

# Или создать заглушки в Makefile
```

### 7. Проблемы с сетью и таймаутами при скачивании зависимостей

**Ошибка:**
```
read: connection timed out
```

**Решение:**
```bash
# Настроить Go прокси
go env -w GOPROXY=https://goproxy.io,direct
go env -w GOSUMDB=off
go env -w GO111MODULE=on
```

### 8. Финальная сборка бинарника

**Решение:**
```bash
# Успешная сборка
go build -mod=vendor -o bin/opencloud ./opencloud/cmd/opencloud

# Проверка
./bin/opencloud --version
# opencloud version (OpenCloud GmbH <support@opencloud.eu>) (2026-05-05)
```

## Запуск OpenCloud

```bash
# Инициализация конфигурации
./bin/opencloud init

# Запуск сервера
./bin/opencloud server
```

## Итоговые замечания

1. **Основная проблема** - проект находится в стадии перехода от OwnCloud к OpenCloud, многие зависимости остались с префиксом `owncloud`

2. **Ключевые инструменты**, которые пришлось устанавливать вручную:
   - buf (для protobuf)
   - pigeon (для парсера грамматики)
   - bingo (для управления зависимостями)

3. **Временное отключение сервисов** (search и settings) не влияет на базовую функциональность файлового хранилища

4. **Успешная сборка** заняла около 2 часов из-за необходимости решения всех проблем

5. **Бинарник** получился размером 150 МБ, что нормально для Go приложения
