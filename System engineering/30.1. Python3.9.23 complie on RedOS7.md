**校褋褌邪薪芯胁泻邪 Python 3.9.23 薪邪 RED OS 7 (芯褋薪芯胁邪薪薪褍褞 薪邪 RHEL/CentOS 7):**
<br/> 褔懈褌邪泄 锌芯写褋泻邪蟹泻褍 [how-can-i-install-python-3-9-on-a-linux-ubuntu-terminal](https://stackoverflow.com/questions/60824700/how-can-i-install-python-3-9-on-a-linux-ubuntu-terminal)

## 1. 校褋褌邪薪芯胁泻邪 蟹邪胁懈褋懈屑芯褋褌械泄

```bash
sudo dnf groupinstall "Development Tools"
sudo dnf install zlib-devel bzip2-devel openssl-devel ncurses-devel sqlite-devel readline-devel tk-devel gdbm-devel db4-devel libpcap-devel xz-devel libffi-devel
```

## 2. 小泻邪褔懈胁邪薪懈械 懈褋褏芯写薪芯谐芯 泻芯写邪 Python 3.9.23

```bash
cd /usr/src
curl https://www.python.org/ftp/python/3.9.23/
sudo wget https://www.python.org/ftp/python/3.9.23/Python-3.9.23.tgz
sudo tar xzf Python-3.9.23.tgz
cd Python-3.9.23
```

## 3. 袣芯薪褎懈谐褍褉邪褑懈褟 懈 泻芯屑锌懈谢褟褑懈褟

```bash
sudo ./configure --enable-optimizations --enable-shared --with-system-ffi --with-computed-gotos --enable-loadable-sqlite-extensions
sudo make -j$(nproc)

# 袛谢褟 褋懈褋褌械屑 褋 /proc/cpuinfo
sudo make -j$(grep -c processor /proc/cpuinfo)

# 袠谢懈 褍泻邪蟹邪褌褜 褎懈泻褋懈褉芯胁邪薪薪芯械 泻芯谢懈褔械褋褌胁芯 (薪邪锌褉懈屑械褉, 4 锌芯褌芯泻邪)
sudo make -j4
```

## 4. 校褋褌邪薪芯胁泻邪

```bash
sudo make altinstall
```

## 5. 袧邪褋褌褉芯泄泻邪 斜懈斜谢懈芯褌械泻

```bash
# 袛芯斜邪胁谢械薪懈械 锌褍褌懈 泻 斜懈斜谢懈芯褌械泻邪屑
echo '/usr/local/lib' | sudo tee /etc/ld.so.conf.d/python3.9.conf
sudo ldconfig

# 袩褉芯胁械褉懈褌褜, 褔褌芯 斜懈斜谢懈芯褌械泻懈 薪邪泄写械薪褘
ldconfig -p | grep libpython3.9

# 袩褉芯胁械褉懈褌褜, 芯褌泻褍写邪 Python 谐褉褍蟹懈褌 斜懈斜谢懈芯褌械泻懈
ldd $(which python3.9) | grep libpython

# 袩褉芯胁械褉懈褌褜, 褔褌芯 Python 蟹邪锌褍褋泻邪械褌褋褟 斜械蟹 芯褕懈斜芯泻
python3.9 -c "print('袘懈斜谢懈芯褌械泻懈 褉邪斜芯褌邪褞褌!')"
```

## 6. 袩褉芯胁械褉泻邪 褍褋褌邪薪芯胁泻懈

```bash
python3.9 --version
which python3.9
```

## 7. 小芯蟹写邪薪懈械 邪谢褜褌械褉薪邪褌懈胁薪芯泄 褋褋褘谢泻懈 (芯锌褑懈芯薪邪谢褜薪芯)

```bash
sudo alternatives --install /usr/bin/python3 python3 /usr/local/bin/python3.9 1
sudo alternatives --config python3
```

## 8. 校褋褌邪薪芯胁泻邪 pip 写谢褟 Python 3.9

```bash
python3.9 -m ensurepip --upgrade
python3.9 -m pip install --upgrade pip
```

## 袙邪卸薪褘械 蟹邪屑械褔邪薪懈褟:

1. **`make altinstall`** 胁屑械褋褌芯 `make install` - 褔褌芯斜褘 薪械 锌械褉械蟹邪锌懈褋褘胁邪褌褜 褋懈褋褌械屑薪褘泄 Python
2. **`--enable-shared`** - 写谢褟 褋芯蟹写邪薪懈褟 shared 斜懈斜谢懈芯褌械泻
3. **`--enable-optimizations`** - 写谢褟 芯锌褌懈屑懈蟹邪褑懈懈 锌褉芯懈蟹胁芯写懈褌械谢褜薪芯褋褌懈
4. RED OS 芯褋薪芯胁邪薪邪 薪邪 RHEL 7, 锌芯褝褌芯屑褍 卸械谢邪褌械谢褜薪芯 懈褋锌芯谢褜蟹芯胁邪褌褜 `yum` 胁屑械褋褌芯 `dnf`
5. 袩褉芯胁械褉褜褌械 写芯褋褌褍锌薪芯械 屑械褋褌芯 薪邪 写懈褋泻械 锌械褉械写 泻芯屑锌懈谢褟褑懈械泄 (褌褉械斜褍械褌褋褟 ~2-3GB)

袩芯褋谢械 褍褋褌邪薪芯胁泻懈 褍 胁邪褋 斜褍写械褌:
- Python 3.9.23 写芯褋褌褍锌薪褘泄 泻邪泻 `python3.9`
- pip 写谢褟 Python 3.9
- 小芯胁屑械褋褌懈屑芯褋褌褜 褋 褋褍褖械褋褌胁褍褞褖懈屑懈 褋懈褋褌械屑薪褘屑懈 锌邪泻械褌邪屑懈
