# Шпаргалка по командам pveam

| Команда | Описание |
|---------|----------|
| `pveam update` | Обновить список доступных шаблонов |
| `pveam available` | Показать все доступные шаблоны |
| `pveam available --section system` | Показать только системные шаблоны |
| `pveam download <storage> <template>` | Скачать шаблон в указанное хранилище |
| `pveam list` | Показать все шаблоны в локальных хранилищах |
| `pveam list <storage>` | Показать шаблоны в конкретном хранилище |
| `pveam remove <template>` | Удалить шаблон из хранилища |
| `pveam help` | Показать справку по командам |

## Примеры использования:

```bash
# Обновить список шаблонов
pveam update

# Посмотреть все доступные шаблоны Ubuntu
pveam available | grep ubuntu

# Скачать Ubuntu 25.04 и др. ОС в хранилище 'local'
pveam download local debian-13-standard_13.1-2_amd64.tar.zst
pveam download local ubuntu-25.04-standard_25.04-1.1_amd64.tar.zst
pveam download local gentoo-current-openrc_20250508_amd64.tar.xz
pveam download local archlinux-base_20240911-1_amd64.tar.zst

# Показать все скачанные шаблоны
pveam list local

# Удалить шаблон
pveam remove local:ubuntu-22.04-standard_22.04-1_amd64.tar.zst
```

**`смотри в /var/lib/vz/template/cache/`**

## Дополнительно: управление хранилищами и шаблонами

### Просмотр всех хранилищ и их статуса
```bash
pvesm status
```

### Просмотр конфигурации хранилищ
```bash
cat /etc/pve/storage.cfg
```

### Особенности работы с несколькими хранилищами

В Proxmve может быть настроено несколько хранилищ для шаблонов (тип `dir`). Например:

| Хранилище | Реальный путь | Назначение |
|-----------|---------------|------------|
| `local` | `/var/lib/vz/template/cache/` | Стандартное хранилище |
| `home` | `/home/template/cache/` | Дополнительное хранилище |
| `ssd_1tb` | `/stg/1tb` | Быстрое SSD-хранилище |

**Важно:** команда `pveam list` без аргумента **не работает** — всегда нужно указывать имя хранилища:
```bash
pveam list home      # шаблоны в /home/template/cache/
pveam list local     # шаблоны в /var/lib/vz/template/cache/
pveam list ssd_1tb   # шаблоны в /stg/1tb
```

### Поиск и удаление старых шаблонов

```bash
# Посмотреть все шаблоны в хранилище
pveam list home

# Удалить конкретный шаблон
pveam remove home:debian-11-standard_11.7-1_amd64.tar.zst

# Удалить все шаблоны из хранилища (осторожно!)
pveam list home | awk '{print $1}' | xargs -I {} pveam remove home:{}
```

### Где искать файлы шаблонов

Путь к файлам зависит от конфигурации хранилища в `/etc/pve/storage.cfg`:

```bash
# Для хранилища 'local' — стандартный путь
ls -la /var/lib/vz/template/cache/

# Для хранилища 'home' — путь из конфига (если указан path /home)
ls -la /home/template/cache/
```
