## Инструкция по установке Python 3.9.17 как системного на Debian 13

**Требование** - **Скомпилировать и сделать системным Python 3.9.17** 

### Шаг 1: Установка зависимостей
```bash
sudo apt update
sudo apt install -y wget build-essential libssl-dev zlib1g-dev \
libbz2-dev libreadline-dev libsqlite3-dev libncursesw5-dev \
xz-utils tk-dev libxml2-dev libxmlsec1-dev libffi-dev liblzma-dev \
llvm libgdbm-dev libnss3-dev libdb-dev uuid-dev \
libc6-dev libpcre2-dev libbluetooth-dev
```

### Шаг 2: Скачивание и подготовка исходников
```bash
cd /usr/src
sudo wget https://www.python.org/ftp/python/3.9.17/Python-3.9.17.tgz
sudo tar -xzf Python-3.9.17.tgz
cd Python-3.9.17
```

### Шаг 3: Конфигурация с приоритетом системной установки
```bash
sudo ./configure --enable-optimizations \
--prefix=/usr \
--with-openssl=/usr \
--enable-shared \
--with-system-expat \
--with-system-ffi \
--with-ensurepip=install \
LDFLAGS="-Wl,-rpath /usr/lib:/usr/lib/x86_64-linux-gnu"
```

### Шаг 4: Компиляция
```bash
sudo make -j $(nproc)
```

### Шаг 5: Установка
```bash
sudo make install
```

### Шаг 6: Создание ссылок для pip
```bash
sudo ln -sf /usr/bin/pip3.9 /usr/bin/pip3
sudo ln -sf /usr/bin/pip3.9 /usr/bin/pip
```

### Шаг 7: Настройка приоритетов через update-alternatives
```bash
# Добавляем Python 3.9 с высоким приоритетом
sudo update-alternatives --install /usr/bin/python python /usr/bin/python3.9 100
sudo update-alternatives --install /usr/bin/python3 python3 /usr/bin/python3.9 100

# Настраиваем автоматический выбор
sudo update-alternatives --set python /usr/bin/python3.9
sudo update-alternatives --set python3 /usr/bin/python3.9
```

### Шаг 8: Проверка и тестирование
```bash
python3 --version
# Должно быть: Python 3.9.17

python --version  
# Должно быть: Python 3.9.17

pip3 --version
# Должно показывать pip для Python 3.9.17
```

### Шаг 9: Установка pip и обновление (если нужно)
```bash
python3 -m pip install --upgrade pip setuptools wheel
```

### Шаг 10: Проверка работы системных утилит
```bash
# Проверяем, что apt работает
apt --version

# Проверяем другие системные утилиты
lsb_release -a
```



## Важные замечания для Debian 13:

1. **Проблемы с apt**: Если apt перестанет работать, исправляем:
```bash
sudo apt --fix-broken install
# Или временно используем python3.13
sudo update-alternatives --config python3
# Выбираем /usr/bin/python3.13
```

2. **Сохранение совместимости**: Сохраняем ссылку на старый Python:
```bash
sudo ln -sf /usr/bin/python3.13 /usr/bin/python3.13-backup
```

3. **Если что-то пошло не так** - восстановление:
```bash
sudo update-alternatives --remove python /usr/bin/python3.9
sudo update-alternatives --remove python3 /usr/bin/python3.9
sudo apt install --reinstall python3 python3-minimal
```

## Проверка для Jenkins:
```bash
# Проверяем, что Jenkins сможет использовать Python
python3 -c "import sys; print(sys.version)"
# Python 3.9.17

# Устанавливаем необходимые для Jenkins пакеты
pip3 install pytest pylint coverage

# обновление pip
pip install --upgrade pip
```

```bash
sudo reboot

exec bash -l
```
