# Кластеризация Teleport 


## Варианты кластеризации

| Вариант | Сложность | Для каких задач |
|---------|-----------|-----------------|
| **Только Proxy HA** | Низкая | Балансировка веб-трафика (у тебя уже один Proxy — можно добавить ещё) |
| **Auth HA** | Средняя | Отказоустойчивость центра аутентификации (требует внешнюю БД) |
| **Полный кластер** | Высокая | Enterprise-сценарии, большие нагрузки |
