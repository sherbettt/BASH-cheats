### 📋 Стандартные команды по умолчанию

Вот список полезных комбинаций, которые работают "из коробки":

| Действие | Комбинация клавиш |
| :--- | :--- |
| **Отмена (Undo)** | `Ctrl+u` |
| **Вставка (Paste)** | `Shift+Insert` |
| **Копирование (Copy)** | `Ctrl+Insert` |
| **Вырезание (Cut)** | `Shift+Delete` |
| **Поиск** | `F7` |
| **Повтор поиска** | `Shift+F7` |
| **Замена** | `F4` |
| **Удаление строки** | `Ctrl+y` |
| **Выделить блок** | `F3` (начало/конец) |
| **Копировать блок** | `F5` |
| **Переместить блок** | `F6` |
| **Сохранить файл** | `F2` |
| **Сохранить как...** | `Shift+F2` |
| **Вставить файл** | `Shift+F5` |
| **Вызов меню** | `F9` |
| **Выход** | `F10` |

### ⚙️ Настройка горячих клавиш

В `mcedit` нет привычных `Ctrl+Z` (отмена) и `Ctrl+V` (вставка) по умолчанию, но их можно переназначить. Вот ключевая информация.

Чтобы добавить привычные сочетания, нужно создать файл `~/.config/mc/mc.keymap` со следующим содержимым:

```ini
[editor]
Undo = ctrl-z          # Отмена действия
Paste = ctrl-v         # Вставка из буфера
Copy = ctrl-c          # Копирование
Cut = ctrl-x           # Вырезание
MarkAll = ctrl-a       # Выделить всё
Save = ctrl-s          # Сохранить
```
> Учтите, что в терминале сочетание `Ctrl+S` может блокировать вывод. Чтобы отключить это поведение, используйте команду `stty -ixon`.

В этом файле вы можете переназначить любые другие команды. После создания файла, возможно, потребуется перезапустить `mc`.
