## Инструкция по подключению к MikroTik CRS326-24G-2S+RM

### Шаг 1: Проверка наличия Ethernet-адаптера

Выполните в терминале:
```bash
ip link show
```

Вы должны увидеть что-то вроде:
```
1: lo: <LOOPBACK,UP,LOWER_UP> ...
2: wlan0: <BROADCAST,MULTICAST,UP,LOWER_UP> ...
3: enp0s25: <BROADCAST,MULTICAST> ...  # или eth0
```

Если вы **не видите** интерфейса с названием `eth`, `enp` или `eno`, то:

**Вариант А:** Убедитесь, что Ethernet-кабель физически подключен от компьютера к MikroTik (в любой порт 1-24).
**Вариант Б:** На некоторых ноутбуках Ethernet-адаптер может быть отключен в BIOS или программно. Проверьте его статус:
```bash
rfkill list
```
Если Ethernet заблокирован, разблокируйте:
```bash
sudo rfkill unblock all
```

После подключения кабеля выполните `ip link show` снова. Должен появиться новый интерфейс (например, `enp0s25` или `eth0`).

---

### Шаг 2: Создание нового проводного подключения через nmtui

Поскольку используется NetworkManager, мы создадим новое статическое подключение для Ethernet.

1. **Запустите текстовый интерфейс NetworkManager:**
   ```bash
   nmtui
   ```

2. **Выберите "Edit a connection"** (стрелками, Enter)

3. **Нажмите "Add"** (кнопка внизу, Tab для переключения)

4. **Выберите тип "Ethernet"** и нажмите Enter

5. **Заполните поля:**
   - **Profile name:** `MikroTik` (или любое имя)
   - **Device:** выберите ваш Ethernet-интерфейс (например, `enp0s25` или тот, который появился после подключения кабеля)
   - **IPv4 CONFIGURATION:** измените с `<Automatic>` на **`<Manual>`**
   - Нажмите **"Show"** напротив пункта, который появится ниже
   - **Addresses:** введите `192.168.88.2/24`
   - **Gateway:** введите `192.168.88.1`
   - **DNS servers:** введите `8.8.8.8, 1.1.1.1`

6. **Нажмите "OK"** внизу, затем **"Back"** и **"Quit"**

---

### Шаг 3: Подключение к MikroTik

1. **Отключите Wi-Fi** (чтобы не было конфликта маршрутов):
   ```bash
   nmcli connection down runtel
   ```

2. **Подключите созданное Ethernet-подключение:**
   ```bash
   nmcli connection up MikroTik
   ```
   (Если вы назвали подключение иначе, используйте это имя)

3. **Проверьте, что IP назначился:**
   ```bash
   ip addr show
   ```
   Вы должны увидеть ваш Ethernet-интерфейс с IP `192.168.88.2/24`

4. **Проверьте связь с коммутатором:**
   ```bash
   ping 192.168.88.1
   ```
   Должны появиться ответы:
   ```
   PING 192.168.88.1 (192.168.88.1) 56(84) bytes of data.
   64 bytes from 192.168.88.1: icmp_seq=1 ttl=64 time=0.5 ms
   ```

---

### Шаг 4: Запуск WinBox для настройки MikroTik

1. **Установите Wine** (если еще не установлен):
   ```bash
   sudo pacman -S wine
   ```

2. **Перейдите в папку с WinBox:**
   ```bash
   cd ~/Programs/WinBox-Mikrotik
   ```

3. **Запустите WinBox:**
   ```bash
   wine winbox.exe
   ```
   (или `wine winbox64.exe`, если скачали 64-битную версию)

4. **В окне WinBox:**
   - **Connect To:** `192.168.88.1`
   - **Login:** `admin`
   - **Password:** оставьте пустым (или введите пароль с наклейки на корпусе, если есть)
   - Нажмите **Connect**

---

### Шаг 5: Что делать, если WinBox не запускается или нет ответа на ping

**Если ping не проходит:**

1. **Убедитесь, что MikroTik включен** — на передней панели должны гореть индикаторы портов.
2. **Проверьте порт** — подключите компьютер к любому порту 1-24, но **не** к SFP+ портам (25 и 26).
3. **Сбросьте MikroTik до заводских настроек:**
   - Отключите питание коммутатора
   - Зажмите кнопку **Reset** (обычно на передней панели)
   - Включите питание, удерживая кнопку
   - Дождитесь мигания LED (обычно через 5-10 секунд) и отпустите
   - Подождите 1-2 минуты, пока устройство загрузится

**Если WinBox не запускается через Wine:**

Попробуйте использовать **WebFig** — откройте браузер (Firefox/Chrome) и перейдите по адресу:
```
http://192.168.88.1
```
Это полный веб-интерфейс MikroTik, который работает без Wine.

---

### Шаг 6: После успешного входа

1. **Смените пароль администратора:**
   - WinBox: **System → Password**
   - WebFig: **System → Password**

2. **Обновите RouterOS:**
   - **System → Packages → Check for Updates**

3. **Настройте режим работы** (если нужно):
   - **System → RouterBOARD → Settings**
   - Boot OS: RouterOS (полнофункциональный) или SwOS (только коммутатор)

---

### Шаг 7: Возврат к обычной сети (после настройки)

После завершения настройки MikroTik вы можете вернуться к Wi-Fi:

```bash
nmcli connection down MikroTik
nmcli connection up runtel
```
