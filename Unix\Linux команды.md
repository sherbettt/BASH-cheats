# Проверка условий в bash

## Проверка строк:

- `[ -z "$STRING" ]` - строка пустая
- `[ -n "$STRING" ]` - строка не пустая

## Проверка файлов:

- `[ -f "$FILE" ]` - файл существует и это обычный файл
- `[ -d "$DIR" ]` - директория существует
- `[ -r "$FILE" ]` - файл существует и доступен для чтения
- `[ -w "$FILE" ]` - файл существует и доступен для записи
- `[ -x "$FILE" ]` - файл существует и доступен для выполнения

## Числовые сравнения:

- `[ $NUM -eq $NUM2 ]` - равно
- `[ $NUM -ne $NUM2 ]` - не равно
- `[ $NUM -lt $NUM2 ]` - меньше
- `[ $NUM -gt $NUM2 ]` - больше

## Пример для вашего случая

Правильная проверка существования бекап-файла:

```bash
if [ ! -f "$BACKUP_FILE" ]; then
    echo "Бекап не найден"
    # действия при отсутствии бекапа
fi
```

**Примечание:** Знак `!` означает отрицание (NOT), то есть "если файл НЕ существует".

### **Самый быстрый способ - терминал:**
```bash
# Посмотреть справку по всем тестовым операторам
man test

# Или короче
help test

# Для bash-специфичных выражений
man bash
# Затем внутри мануала ищите "CONDITIONAL EXPRESSIONS"

# Альтернативно
info test
```

### **Проверить работу прямо в терминале:**
```bash
# Протестировать прямо в командной строке
test -f /etc/passwd && echo "Файл существует"

# Или с квадратными скобками (это синоним команды test)
[ -f /etc/passwd ] && echo "Файл существует"
```

## **3. Практические места для изучения**

### **Для начинающих:**
```bash
# Создайте тестовый скрипт
nano test_script.sh

# Содержимое:
#!/bin/bash
FILE="/etc/passwd"

if [ -f "$FILE" ]; then
    echo "Файл $FILE существует"
fi

if [ -r "$FILE" ]; then
    echo "И доступен для чтения"
fi
```

### **Быстрые примеры для запоминания:**
```bash
# Проверка в одну строку
[ -f "/etc/passwd" ] && echo "Существует" || echo "Не существует"

# Проверка директории
[ -d "/tmp" ] && echo "Директория /tmp существует"
```
