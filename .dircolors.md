### Создать свой конфиг
```bash
dircolors -p > ~/.dircolors
```

**Вариант 1 - Ярко-голубой (рекомендую):**
```bash
DIR 01;96		# самый яркий и заметный
```

**Вариант 2 - Ярко-зелёный:**
```bash
DIR 01;92		# очень яркий зелёный
```

**Вариант 3 - Ярко-пурпурный:**
```bash
DIR 01;95		# яркий фиолетовый
```

**Вариант 4 - Неоново-оранжевый:**
```bash
DIR 01;38;5;208	# оранжевый как в твоём PS1
```

**Вариант 5 - Ярко-синий:**
```bash
DIR 01;94		# яркий синий
```

## Яркие коды цветов:
- `01;91` - ярко-красный
- `01;92` - ярко-зелёный  
- `01;93` - ярко-жёлтый
- `01;94` - ярко-синий
- `01;95` - ярко-пурпурный
- `01;96` - ярко-голубой
- `01;97` - ярко-белый

**Применить:**
```bash
eval "$(dircolors ~/.dircolors)"
```
