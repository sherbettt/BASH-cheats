## 1. Просмотр доступных WiFi сетей

```bash
# Простой список сетей
nmcli device wifi list

# Более подробная информация
nmcli device wifi list --rescan yes

# Список с дополнительными деталями
nmcli -f SSID,BSSID,SIGNAL,SECURITY device wifi list
```

## 2. Подключение к WiFi сети

### Способ 1: Подключение с паролем в команде
```bash
nmcli device wifi connect "Имя_сети" password "ваш_пароль"
```

### Способ 2: Подключение с запросом пароля
```bash
nmcli device wifi connect "Имя_сети" --ask
```

### Способ 3: Если сеть скрыта
```bash
nmcli device wifi connect "Имя_сети" password "ваш_пароль" hidden yes
```

### Способ 4: Подключение к конкретной точке доступа по BSSID (MAC-адресу)
Когда в зоне видимости несколько точек доступа с одинаковым именем (SSID), вы можете выбрать конкретную по её BSSID:

```bash
# Подключиться к сети с указанием конкретного BSSID
nmcli device wifi connect "Имя_сети" password "ваш_пароль" bssid AA:BB:CC:DD:EE:FF
```

Пример из реального сканирования:
```bash
# Сначала смотрим список сетей с BSSID
nmcli -f SSID,BSSID,SIGNAL device wifi list

# Подключаемся к конкретной точке доступа runtel (например, на 2.4 ГГц)
nmcli device wifi connect "runtel" password "мой_пароль" bssid 48:22:54:08:BF:26

# Или к точке на 5 ГГц
nmcli device wifi connect "runtel" password "мой_пароль" bssid BC:07:1D:FB:52:1B
```

### Способ 5: Подключение используя только BSSID (без SSID)
```bash
# Если вы знаете только MAC-адрес точки доступа
nmcli device wifi connect AA:BB:CC:DD:EE:FF password "ваш_пароль"
```

## 3. Управление подключениями и профилями

```bash
# Показать все сохраненные подключения
nmcli connection show

# Показать только активные подключения
nmcli connection show --active

# Показать статус сетевых устройств
nmcli device status

# Подключиться к сохраненной сети
nmcli connection up "Имя_сети"

# Отключиться от текущей сети
nmcli device disconnect wlan0  # замените wlan0 на ваш интерфейс

# Забыть (удалить) сохраненную сеть
nmcli connection delete "Имя_сети"

# Переключиться на другую точку доступа с тем же SSID
nmcli device wifi connect "Имя_сети" bssid AA:BB:CC:DD:EE:FF
```

## 4. Работа с несколькими точками доступа (одинаковый SSID)

```bash
# Посмотреть все точки доступа с одинаковым именем
nmcli -f SSID,BSSID,CHAN,SIGNAL device wifi list | grep "runtel"

# Создать отдельный профиль для конкретной точки доступа
nmcli connection add type wifi con-name "runtel-5GHz" ifname wlan0 ssid "runtel"
nmcli connection modify "runtel-5GHz" wifi.bssid BC:07:1D:FB:52:1B
nmcli connection modify "runtel-5GHz" wifi-sec.key-mgmt wpa-psk
nmcli connection modify "runtel-5GHz" wifi-sec.psk "ваш_пароль"
nmcli connection up "runtel-5GHz"
```

## 5. Пример полного процесса

```bash
# Сканировать и показать сети
nmcli device wifi list

# Подключиться к выбранной сети (обычный способ)
nmcli device wifi connect "MyWiFiNetwork" password "mypassword123"

# Или подключиться к конкретной точке доступа по BSSID
nmcli device wifi connect "MyWiFiNetwork" password "mypassword123" bssid 48:22:54:08:BF:26

# Проверить статус подключения
nmcli connection show --active

# Узнать, через какую точку доступа (BSSID) вы сейчас подключены
iwconfig wlan0 | grep "Access Point"
# или
nmcli -f GENERAL,WIFI-PROPERTIES device show wlan0
```

## 6. Дополнительные опции

```bash
# Принудительное сканирование сетей
nmcli device wifi rescan

# Показать информацию о конкретной сети
nmcli device wifi list ifname wlan0  # для конкретного интерфейса

# Сохранить подключение для автоматического соединения
nmcli connection modify "Имя_сети" connection.autoconnect yes

# Заблокировать подключение только к определенному BSSID
nmcli connection modify "Имя_сети" wifi.bssid AA:BB:CC:DD:EE:FF

# Посмотреть детальную информацию о текущем подключении
nmcli connection show "Имя_сети"
```

## 7. Диагностика и устранение проблем

```bash
# Проверить состояние WiFi радио
nmcli radio wifi

# Включить WiFi адаптер
nmcli radio wifi on

# Отключить WiFi адаптер
nmcli radio wifi off

# Посмотреть общую статистику
nmcli general status

# Подробная информация об интерфейсе
nmcli device show wlan0

# Просмотр логов подключения
journalctl -u NetworkManager -f

# Тест скорости соединения с точкой доступа
sudo iw dev wlan0 link
```

## Важные моменты:

- Убедитесь, что WiFi адаптер включен: `nmcli radio wifi on`
- Если возникают проблемы, проверьте: `nmcli general status`
- Для некоторых операций может потребоваться `sudo`
- BSSID (MAC-адрес) точки доступа можно узнать из вывода `nmcli device wifi list`
- Если в офисе несколько точек доступа с одинаковым именем, подключение по BSSID поможет выбрать конкретную (например, ближайшую или менее загруженную)
- При подключении по BSSID убедитесь, что вы находитесь в зоне уверенного приема сигнала от этой конкретной точки
