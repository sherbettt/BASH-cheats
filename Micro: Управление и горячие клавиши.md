## 🎮 Управление и горячие клавиши в Micro

Micro позиционирует себя как «современный терминальный текстовый редактор». 
Он дружелюбен к новичкам, потому что использует **привычные сочетания Ctrl** (как в блокнотах и IDE), а не модальную систему, как Vim.

### Базовые (самые нужные)
| Сочетание | Действие |
| :--- | :--- |
| `Ctrl + S` | Сохранить файл |
| `Ctrl + Q` | Закрыть текущую вкладку (или выйти, если вкладка одна) |
| `Ctrl + C / V / X` | Копировать / Вставить / Вырезать |
| `Ctrl + Z` | Отменить действие |
| `Ctrl + Y` | **Вернуть** действие (отмена отмены) |
| `Ctrl + A` | Выделить всё |
| `Ctrl + F` | Найти в файле. Для перехода к следующему совпадению — `Ctrl + N`, к предыдущему — `Ctrl + P` |
| `Ctrl + G` | Открыть **встроенную справку** (откроется прямо в редакторе) |

### Работа с вкладками и окнами
| Сочетание | Действие |
| :--- | :--- |
| `Ctrl + T` | Создать новую пустую вкладку |
| `Alt + ,` и `Alt + .` | Переключаться между вкладками (влево/вправо) |
| `Ctrl + \` | Разделить окно **по вертикали** (две панели рядом) |
| `Ctrl + _` | Разделить окно **по горизонтали** (панели друг под другом) |
| `Ctrl + W` | Переключить фокус между разделёнными панелями |

### Продвинутые команды (через командную строку)
Нажмите `Ctrl + E` (или `Alt + E`), чтобы открыть командную строку внизу экрана, и введите:

| Команда | Действие |
| :--- | :--- |
| `save` | Сохранить файл |
| `saveas ~/file.txt` | Сохранить с новым именем |
| `quit` или `q` | Закрыть текущую вкладку |
| `qall` | Выйти из редактора, закрыв **все** вкладки |
| `open file.go` | Открыть файл в текущей вкладке |
| `tabnew main.go` | Открыть файл в **новой** вкладке |
| `vsplit file.txt` | Разделить окно вертикально и открыть там файл |
| `hsplit file.txt` | Разделить окно горизонтально и открыть там файл |

## 🎨 Подсветка синтаксиса и цветовые схемы

Micro "из коробки" поддерживает **более 130 языков**. Он автоматически определяет язык по расширению файла (`.py`, `.go`, `.js`, `.md` и т.д.).

### Как сменить тему (цветовую схему)?
1. Нажмите `Ctrl + E`.
2. Введите команду:  
   `set colorscheme название_темы`

**Популярные темы:**
- `default` (стандартная)
- `twilight` (тёплая тёмная)
- `solarized` (глазам приятно)
- `monokai` (как в Sublime Text)
- `simple` (минималистичная, без 256 цветов)

> ⚠️ **Важно:** Если подсветка не работает или цвета "битые", ваш терминал может не поддерживать 256 цветов.  
> Попробуйте: `set colorscheme simple`  
> Или проверьте переменную терминала: `export TERM=xterm-256color` (добавьте в `.bashrc` или `.zshrc`).

### Как включить/выключить подсветку?
Отключить совсем:
```bash
set syntax off
```
Включить обратно:
```bash
set syntax on
```

## 🖱️ Дополнительные возможности

### Поддержка мыши
Мышь работает по умолчанию. Вы можете:
- **Кликнуть** — поставить курсор.
- **Перетаскивать** — выделить текст.
- **Двойной клик** — выделить слово.
- **Тройной клик** — выделить всю строку.

### Множественные курсоры (Multi-cursor)
Это "киллер-фича" для замены одинаковых слов в разных местах:
- Удерживайте `Ctrl` и кликайте мышкой в нужных местах — появятся дополнительные курсоры.
- Или зажмите `Ctrl + Alt` и стрелками вверх/вниз, чтобы добавить курсоры на соседние строки.

## ⚙️ Как переназначить клавиши (Редактирование `bindings.json`)

Вы не ограничены стандартными сочетаниями. Файл настроек лежит тут:  
`~/.config/micro/bindings.json`

**Пример содержимого:**
```json
{
    "Alt-s": "Save,Quit",
    "Ctrl-Alt-F": "Find",
    "F1": "help"
}
```
- Чтобы **отключить** стандартную привязку, поставьте ей значение `None`.
- Чтобы привязать **цепочку команд**, пишите через запятую (без пробелов).

## 🧩 Плагины (расширения)

В Micro встроен простой менеджер плагинов. Команды выполняются всё там же, через `Ctrl + E`:

| Команда | Действие |
| :--- | :--- |
| `plugin install go` | Установить плагин для Go (автодополнение, линтинг) |
| `plugin install git` | Добавить интеграцию с Git (показывает изменения) |
| `plugin remove git` | Удалить плагин |
| `plugin list` | Показать список установленных |

> Плагины устанавливаются в `~/.config/micro/plugins/`

## 🛠️ Где хранятся настройки и как их менять

Все конфиги лежат в папке `~/.config/micro/`:

- `settings.json` — основные настройки редактора.
- `bindings.json` — привязка клавиш.
- `colorschemes/` — папка с вашими кастомными темами.

**Полезные настройки (в `settings.json`):**
```json
{
    "tabsize": 4,
    "tabstospaces": true,
    "ruler": true,
    "cursorline": true,
    "mouse": true,
    "autosu": true
}
```
Изменить их можно двумя способами:
1. Через команду в `Ctrl + E`: `set tabsize 2`
2. Прямо в файле `settings.json`.

## ❓ Если что-то пошло не так

- **Справка прямо в редакторе** — нажмите `Ctrl + G` (или введите `help` в командной строке).
- **Список всех горячих клавиш** — нажмите `Alt + G`.
- **Посмотреть текущие настройки** — введите `set` (без параметров) в командной строке.
