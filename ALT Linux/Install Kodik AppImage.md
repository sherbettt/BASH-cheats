### Способ 1: Установить libfuse2 (наиболее правильный)

Поскольку вы используете дистрибутив на основе Debian/Ubuntu (судя по командам), выполните в терминале:

```bash
sudo apt update
sudo apt install libfuse2
```

После установки попробуйте запустить снова:
```bash
./Kodik-1.4.0.AppImage
```

### Способ 2: Извлечь содержимое (без установки FUSE)

Если нет прав на установку программ, можно просто распаковать AppImage и запустить файл внутри:

```bash
# Извлечение в папку squashfs-root
./Kodik-1.4.0.AppImage --appimage-extract

# Запуск приложения
./squashfs-root/AppRun
```

После этого папку `squashfs-root` можно переместить куда угодно и запускать `AppRun` оттуда.

### Способ 3: Использовать флаг `--no-sandbox` (если это Electron-приложение)

Некоторые AppImage на основе Electron требуют отключения песочницы, если FUSE не работает:

```bash
./Kodik-1.4.0.AppImage --no-sandbox
```
